Tags


As tags são usadas para informar ao navegador como o site deve ser apresentado. Todas as tags possuem um padrão de codificação, ou seja, um formato padrão que deve ser seguido pelo desenvolvedor. Elas começam com o sinal "<", e terminam com o sinal ">". Existem dois tipos de tag: tags de abertura e tags de fechamento. A principal diferença na escrita é uma barra (/) para fechar a marcação. Deve-se abrir a tag, inserir o conteúdo desejado, e a tag é fechada.

Tag "html"

A tag html é conhecida como o "elemento raiz" do documento. Todas as outras tags são descendentes deste elemento. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tag "head"

A tag head é usada como cabeçalho do documento, ou seja, serve para introduzir e explicitar informações sobre o documento. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tag "body"

A tag body, como o nome ja diz, é o corpo do documento. Dentro dela, serão colocadas as tags que irão compor a página desenvolvida. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tag "h1"

A tag h1 é usada para diferenciar o título principal do restante do conteúdo. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tags h2 até h6

Essas tags são usadas para diferenciar o conteúdo por importância, facilitando o entendimento na leitura da página. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tag "br"

A tag br é usada para realizar a quebra de uma linha, sendo assim, o texto que for escrito depois desta tag, irá automaticamente para a próxima linha. A tag não precisa ser fechada, a linha será quebrada automaticamente.


Tag "p"

A tag p é utilizada para representar um parágrafo do texto. Ao utilizá-la, é feita uma quebra de linha, e é adicionado um parágrafo. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tag "strong"

A tag strong é utilizada para dar importância ao termo que está entre ela, colocando-o em negrito. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tag "em"

A tag em é utilizada para dar ênfase ao termo que está entre ela, deixando-o automaticamente, em itálico. A tag precisa ser aberta, o conteúdo será digitado, e a tag é fechada.


Tag "font"

A tag font é usada para modificar o tamanho, cor, ou até a face da fonte. Ao digitar a tag, você poderá escolher o que modificar. Digitando "color", você modificará a cor da fonte. Digitando "size", você modificará o tamanho da fonte. Digitando "face", você modificará a face da fonte.


Tag "ul"

A tag ul é utilizada para adicionar uma lista desordenada, ou seja, lista em que a ordenação numérica e suas posições não importam. Os elementos da lista devem estar depois da abertura e antes do fechamento da tag.


Tag "ol"

A tag ol é utilizada para a marcação de listas ordenadas, ou seja,respeitando a ordem e a numeração dos itens da lista. Os elementos da lista devem estar depois da abertura e antes do fechamento da tag.


Tag "figure"

A tag figure é usada para marcar conteúdos ilustrativos, como por exemplo, imagens,fotos e diagramas.


Tag "img"

Utilizamos esta tag para inserir imagens. O atributo src define o arquivo da imagem, o atributo alt a descreve, e os atributos height e width te permitem ajustar largura e altura da imagem. A tag precisa ser aberta, porém não necessita de fechamento.


Tag "figcaption"

Essa tag permite adicionar uma legenda a imagem.
